HLS Player для GitHub Pages

Решение для воспроизведения защищённых потоков с обходом ограничений через прокси

Выберите прокси-сервер для запуска потока

Почему это работает на GitHub Pages?

При публикации на GitHub Pages возникают следующие проблемы:

// 1. Смешанное содержимое: HTTPS страница пытается загрузить HTTP поток
// 2. Ограничения публичных прокси-серверов
// 3. CORS политики безопасности

Решение:

✔️ Все запросы идут через HTTPS-прокси

✔️ Прокси-серверы добавляют необходимые CORS-заголовки

✔️ Автоматическое переключение между разными прокси

Технические детали:

Плеер использует Video.js с плагином HTTP Streaming (HLS.js). При выборе прокси происходит:

// 1. Кодирование URL потока
// 2. Формирование проксированного URL
// 3. Установка источника для Video.js
// 4. Обработка возможных ошибок
